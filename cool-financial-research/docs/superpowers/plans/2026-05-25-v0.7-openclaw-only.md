# v0.7 OpenClaw-Only Cleanup Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use `superpowers:subagent-driven-development` or `superpowers:executing-plans` to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Remove the legacy standalone CLI and direct-orchestrator path so the repository is OpenClaw-only, with `scripts/cfr_helper.py` and the importable deterministic helpers as the only supported runtime surface.

**Architecture:** Keep the OpenClaw skill, prompt files, JSON schemas, and deterministic helper modules. Delete the legacy package CLI and direct workflow stack, trim packaging metadata to the OpenClaw helper path, and rewrite docs/tests so nothing suggests a standalone `cool-financial-research` CLI still exists.

**Tech Stack:** Python 3.10+, stdlib `argparse`, Pydantic, pytest, Ruff, mypy, OpenClaw skill metadata and helper scripts.
